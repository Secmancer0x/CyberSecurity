# OSINT-Based Incident Analysis Reports  

This folder contains incident analysis reports created as part of cybersecurity hands-on practice.  
The reports cover different IOC types such as IP addresses, domains, URLs, and file hashes.  

## Notes  
- All IOCs used in these reports are **publicly available threat intelligence data**.  
- These reports are for **educational and portfolio purposes only**.  
- No private or customer-related data is included.  

## Categories Covered  
- Brute force attempts  
- Port scans  
- Malicious file hashes (Trojan, Exploiter)  
- Phishing domains & URLs  
- False positive cases (DNS IPs, clean domains)
